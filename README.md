#关于脚本
脚本用于在Slackware64 上安装32 位运行库。

如果BUG 请提交Issues：

https://github.com/iSpeller/Slackware-Multilib/issues

#安装使用

    git clone https://github.com/iSpeller/Slackware-Multilib.git /tmp/Slackware-Multilib
    cd /tmp/Slackware-Multilib
    sudo ./install.sh
